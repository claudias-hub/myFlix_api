Movie API Documentation

Welcome to the Movie API documentation. This API allows users to access information about movies, directors, and genres.

API Endpoints

Request URL Response Request Body Response Format Example
Get all movies GET /movies Returns a list of all movies (JSON) No body required [ { "title": "The Shawshank Redemption", "director": "Frank Darabont", "genre": "Drama", "year": 1994 }, ... ]
Get a single movie by title GET /movies/:title Returns details about a specific movie (description, genre, director, image, featured) No body required { "title": "Inception", "director": "Christopher Nolan", "genre": "Science Fiction", "year": 2010, }
Get genre details by name GET /genres/:name Returns genre description No body required { "genre": "Drama", "movies": [ { "title": "The Shawshank Redemption", "director": "Frank Darabont", "year": 1994 }, ... ] }
Get director details by name GET /directors/:name Returns director bio, birth year, and death year No body required { "director": "Christopher Nolan", "movies": [ { "title": "Inception", "genre": "Science Fiction", "year": 2010 }, ... ] }
Register a new user POST /users Adds a new user to the database { "username": "new_user", "email": "newuser@example.com", "password": "securepassword" } { "message": "User registered successfully!", "user": { "username": "new_user", "email": "newuser@example.com", "password": "securepassword" } }
Update user info (username) PUT /users/:username Updates user information { "email": "updated_email@example.com", "password": "newpassword" } { "message": "User new_user has been updated.", "user": { "username": "new_user", "email": "updated_email@example.com", "password": "newpassword" } }
Add a movie to user’s favorite list POST /users/:username/movies/:movieID Confirms that a movie was added to favorites No body required { "message": "Movie 'Inception' added to new_user's favorites." }
Remove a movie from user’s favorite list DELETE /users/:username/movies/:movieID Confirms that a movie was removed from favorites No body required { "message": "Movie 'Inception' removed from new_user's favorites." }
Deregister a user DELETE /users/:username Confirms that the user account was deleted No body required { "message": "User new_user has been removed." }
